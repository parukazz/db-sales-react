# Dashboard Analitik Penjualan

Tools yang digunakan:
- React
- Material UI
- Axios (Ext Library)
- Recharts (Ext Library)
- json-server (Ext Library)

Sebuah dashboard Analisa Penjualan 1 halaman yang berisi komponen-komponen dan beberapa fitur, seperti:
- Tabel Penjualan
- Diagram Garis Penjualan
- Diagram Batang komparasi dari penjualan produk
- Statistik atau rangkuman dari semua data yang ditampilkan oleh tabel maupun diagram
- Fitur filter berdasarkan tanggal yang ditentukan
- Fitur pencarian produk secara live
